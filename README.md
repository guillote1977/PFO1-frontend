# Trabajo Práctico - Portafolio Personal

Este proyecto corresponde al Trabajo Práctico Formativo Obligatorio 1 de Desarrollo Web. Se trata de un portafolio personal desarrollado con HTML5 y CSS3, que incluye secciones como presentación personal, proyectos destacados, habilidades técnicas, películas favoritas y un formulario de contacto. El diseño es totalmente responsivo y utiliza Flexbox y Grid para la organización de los elementos.

**Redacta: ¿Qué ventajas encontraste al utilizar Flexbox o Grid en tu proyecto?**
*Flexbox me resultó ideal para alinear elementos en una sola dimensión, como en la barra de navegación y la sección sobre mí. Grid, por su parte, fue perfecto para las secciones de tarjetas y películas, ya que permite crear fácilmente layouts de dos dimensiones y controlar exactamente cómo se distribuyen las tarjetas en filas y columnas, haciendo el código más limpio y mantenible.*
